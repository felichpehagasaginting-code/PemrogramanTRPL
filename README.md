## Ringkasan Proyek

**Platform**: Matrikulasi TRPL
**Target**: Mahasiswa baru TRPL (18-20 tahun, Gen-Z)
**Tujuan**: Platform belajar pemrograman interaktif yang fun, gamified, dan aman
**Teknologi**: Next.js 14, Supabase, Monaco Editor, Piston Sandbox, Vercel
**Timeline**: 8 Minggu
**Launch**: Sebelum hari pertama matrikulasi

## Fitur Utama

- Login SSO kampus
- 7 modul belajar pemrograman (dari algoritma hingga mini project)
- Live code editor berbasis browser (Python)
- Kuis interaktif (pilihan ganda + drag-and-drop)
- Sistem gamifikasi (poin, badge, level, leaderboard)
- Dashboard admin untuk monitoring progres mahasiswa
- Meme integration yang relevan dan fun
- Keamanan skala kampus (OWASP, sandbox terisolasi, enkripsi)

## Quick Start

Dokumen ini adalah fase planning. Untuk memulai development:
1. Baca `prd.md` untuk memahami kebutuhan bisnis
2. Baca `design.md` untuk panduan visual
3. Baca `tech-architecture.md` untuk stack teknologi
4. Baca `agents.md` untuk memahami pembagian tugas
5. Baca `roadmap.md` untuk timeline dan milestone
6. Baca `matrikulasi.md` untuk struktur konten
7. Baca `presentation-flow.md` untuk persiapan presentasi ke stakeholder
