# Matrikulasi TRPL

Platform pembelajaran pemrograman interaktif untuk mahasiswa baru Program Studi Teknologi Rekayasa Perangkat Lunak (TRPL). Dibangun dengan Next.js, dilengkapi live coding Python, kuis interaktif, sistem gamifikasi (XP/level/badge/leaderboard), dan Firebase Auth.

## Fitur

- 🎮 **Gamifikasi** — Poin, level, badge, dan leaderboard untuk motivasi belajar
- 💻 **Live Coding** — Editor Python berbasis Monaco Editor di browser
- 📝 **Kuis Interaktif** — Evaluasi pemahaman tiap modul
- 📊 **Dashboard** — Tracking progress belajar
- 🏆 **Leaderboard** — Kompetisi sehat antar mahasiswa
- 🔐 **Firebase Auth** — Login dengan akun kampus

## Tech Stack

| Stack | Teknologi |
|-------|-----------|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Auth | Firebase Authentication |
| Editor | Monaco Editor |
| Hosting | Firebase Hosting (static export) |
| Styling | CSS Modules + CSS Variables |

## Struktur Project

```
web/              — Aplikasi Next.js
docs/             — Dokumentasi perencanaan (PRD, arsitektur, dll)
```

## Quick Start

```bash
cd web
npm install
npm run dev
```
